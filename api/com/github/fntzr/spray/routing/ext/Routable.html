Routable

trait Routable extends HttpService with HttpMethods with HttpHelpers with Helpers with HttpFormSupport

Trait contatin methods for resourse implementation. With resourse method you might quick create routes for you controller. Also map form information onto Model.

resourse[Controller, Model]

transform to

pathPrefix("model") {
//methods for controller.index
// controller.show
// ...
}

Note: for new method in controller use fresh name.
